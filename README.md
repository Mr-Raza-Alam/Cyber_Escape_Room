# Cyber Escape Room (Microservices Web Application) Python, Flask, Javascript, HTML/CSS

Architected and developed a full-stack technical escape room using a Microservices ecosystem, consisting of 5 independent Flask web servers interacting cohesively.
Implemented decentralized application logic across 4 distinct independent puzzle domains (Cryptography, Network Routing, Circuit Verification).
Designed a centralized Master Admin Portal to handle cross-service authentication, globally monitor participant progress, and orchestrate time-sensitive server routing.
Built resilient fail-safes and session handling to manage concurrent users during an intense, high-pressure competitive technical event.
